# homebrew-zsasdoctor

Homebrew tap for [zsasdoctor](https://github.com/hariharan-rs-19343/zsasdoctor) — SAS Developer Environment Doctor Tool.

## Installation

```bash
brew tap hariharan-rs-19343/zsasdoctor
brew install zsasdoctor
```

## Usage

```bash
zsasdoctor version
```

## Updating

```bash
brew update
brew upgrade zsasdoctor
```

## Uninstallation

```bash
brew uninstall zsasdoctor
brew untap hariharan-rs-19343/zsasdoctor
```
